Справочный центр LivePoll

Добро пожаловать в систему интерактивных опросов. Данное приложение предназначено для взаимодействия зрителей с ведущим в режиме реального времени.

Для зрителей

  • Авторизация: Чтобы голосовать, пройдите быструю регистрацию или войдите под своим аккаунтом.
  • Голосование: Когда ведущий запустит опрос, он появится в правой панели. Выберите один из вариантов.
  • Результаты: После вашего голоса (или обновления данных сокетом) график отобразит текущую статистику всех участников.

Для администраторов

  • Создание опроса: Используйте кнопку «Создать опрос» в нижней панели управления. Введите вопрос и варианты ответов (минимум 2).
  • Управление: Список активных опросов отображается в панели админа. Вы можете удалить любой опрос, нажав красную кнопку «Удалить».
  • Real-time контроль: Все действия мгновенно отображаются у всех подключенных пользователей.

Технические требования

Для корректной работы системы необходим современный браузер (Chrome, Firefox, Edge) и стабильное интернет-соединение для работы WebSockets.
